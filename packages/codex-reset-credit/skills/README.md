# Skills

`codex-reset-credit/` is the primary skill source for this package:

```text
packages/codex-reset-credit/skills/codex-reset-credit/SKILL.md
```

Bundled scripts live next to that `SKILL.md` under:

```text
packages/codex-reset-credit/skills/codex-reset-credit/scripts/
```

The package root `packages/codex-reset-credit/` is also the Claude Code plugin
root. Do not add a duplicate root-level `SKILL.md`.
