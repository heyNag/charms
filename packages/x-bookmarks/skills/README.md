# Skills

`x-bookmarks/` is the primary skill source for this package:

```text
packages/x-bookmarks/skills/x-bookmarks/SKILL.md
```

Bundled scripts, references, and agent UI metadata live next to that `SKILL.md`
under:

```text
packages/x-bookmarks/skills/x-bookmarks/
```

The package root `packages/x-bookmarks/` is also the Claude Code plugin root.
Do not add a duplicate root-level `SKILL.md`.
