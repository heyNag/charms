# Skills

`watch-video/` is the primary skill source for this package:

```text
packages/watch-video/skills/watch-video/SKILL.md
```

Bundled scripts live next to that `SKILL.md` under:

```text
packages/watch-video/skills/watch-video/scripts/
```

The package root `packages/watch-video/` is also the Claude Code plugin root.
Do not add a duplicate root-level `SKILL.md`.
